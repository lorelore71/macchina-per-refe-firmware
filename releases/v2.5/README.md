Release firmware v2.5 per display JC3248W535C / ESP32-S3.

Novita principali:
- bridge HTTP per controllo da simulatore HTML via Wi-Fi;
- endpoint `/status`, `/run`, `/stop`, `/stopAll` compatibili con il banco prova;
- simulatore HTML aggiornato con pannello connessione ESP32;
- default Avvolgi in CCW;
- limite fisso 300 RPM rimosso.

File:
- `firmware-jc3248w535c-v2.5.bin`: firmware applicativo per OTA.
- `firmware-jc3248w535c-v2.5-merged.bin`: immagine completa flash.
- `firmware-jc3248w535c-v2.5-bootloader.bin`: bootloader.
- `firmware-jc3248w535c-v2.5-partitions.bin`: partizioni.
